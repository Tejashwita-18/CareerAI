# CareerAI

An AI-powered resume analyzer that compares resumes with job descriptions and highlights matching and missing skills.

## Features

- Upload PDF resumes
- Upload job descriptions
- Automatic skill extraction
- Resume-job matching
- Match score calculation
- Streamlit web interface

## Tech Stack

- Python
- Streamlit
- pdfplumber

## Upcoming Features

- AI-powered resume feedback (Gemini)
- ATS score
- Skill gap visualization
- RAG-based career assistant
- Interview coach
